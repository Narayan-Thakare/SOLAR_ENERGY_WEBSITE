Basic Concepts

  • 1. What is a Computer?
    Answer

    It is an electronic device made up of hardware and software. It performs tasks such as storing data, handling data, processing data, and retrieving data.

  • 2. What is Hardware?
    Answer

    It is a combination of electronic components to run the software. Example: RAM, ROM, Graphics Card, Processor, etc.

  • 3. What is Software?
    Answer

    Software is a set of instructions that tells the hardware what to do. It includes operating systems, applications, and programs.

  • 4. What is a Low-Level Language?
    Answer

    The language directly understood by the computer or processor is also called a machine-level language. Example: Binary language.

  • 5. What is Assembly Language?
    Answer

    The language which consists of a predefined set of code words that can be understood by the computer using intermediate software known as an Assembler.

  • 6. What is an Assembler?
    Answer

    An Assembler is an intermediate software that acts as a translator to convert assembly-level language into machine-level language.

  • 7. What is a High-Level Language?
    Answer

    The languages that are very close to human understanding, instructable by the programmer to the computer are known as High-Level languages.

  • 8. Types of Intermediate Software
    Answer

    1. Interpreters
    2. Compilers
    3. Assemblers

  • 9. Who Invented Java and What is His Wife's Name?
    Answer

    Java was invented by James Gosling. His wife's name is Barbara.

  • 10. When was Java Invented?
    Answer

    Java was invented in 1995.

  • 11. What is a Platform?
    Answer

    It is a combination of hardware and software on which applications can run.

  • 12. Why are C and C++ Platform Dependent?
    Answer

    When a programmer writes a program using C and C++ languages and saves it with a `.c` extension, it gives the file to the C compiler. The compiler converts the `.c` file into a `.exe` file, which contains native machine code. This native machine-level code is generated by the C compiler, which can only run on its respective native compiler, not on another compiler. Hence, software written using C, C++ languages are platform dependent.

  • 13. What is the Difference Between C++ and Java?
    Answer

    1. C++ compiles directly to machine code, while Java compiles to bytecode for the JVM (Java Virtual Machine).
    2. C++ requires manual memory management, whereas Java has automatic garbage collection.
    3. Java is platform-independent due to the JVM, while C++ needs recompilation for different platforms.
    4. C++ supports pointers, while Java does not.
