👉🏻👉🏻👉🏻 ***Control Flow Statements*** 👈🏻👈🏻

  • 1. What is a control flow statement?
    Answer

    It is a statement used to control the flow of execution in a program. It is known as a control flow statement.

  • 2. Types of control flow statements?
    Answer

    Control flow statements include decision statements, loop statements, and branching statements.

  • 3. What is a decision statement?
    Answer

    A decision statement allows a program to make decisions based on certain conditions. Examples include `if`, `else if`, and `switch` statements.

  • 4. Types of decision statements?
    Answer

    Types of decision statements include `if` statements, `else if` statements, `else` statements, and `switch` statements.
