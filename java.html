  • 12. Why are C and C++ Platform Dependent?
    Answer

    When a programmer writes a program using C and C++ languages and saves it with a `.c` extension, it gives the file to the C compiler. The compiler converts the `.c` file into a `.exe` file, which contains native machine code. This native machine-level code is generated by the C compiler, which can only run on its respective native compiler, not on another compiler. Hence, software written using C, C++ languages are platform dependent.

  • 13. What is the Difference Between C++ and Java?
    Answer

    1. C++ compiles directly to machine code, while Java compiles to bytecode for the JVM (Java Virtual Machine).
    2. C++ requires manual memory management, whereas Java has automatic garbage collection.
    3. Java is platform-independent due to the JVM, while C++ needs recompilation for different platforms.
    4. C++ supports pointers, while Java does not.

  • 4. What is a static block?

  • 5. In a static block, what kind of name can we set?

    // Superclass
    class Employee {
        String name;
        int id;

            this.name = name;
            this.id = id;
        }
    
        void getDetails() {
            System.out.println("Name: " + name);
            System.out.println("ID: " + id);
        }
    }
    
    // Subclass 1
    class Developer extends Employee {
        String programmingLanguages;
    
        Developer(String name, int id, String programmingLanguages) {
            super(name, id);
            this.programmingLanguages = programmingLanguages;
        }
    
        void writeCode() {
            System.out.println(name + " is writing code in: " + programmingLanguages);
        }
    }
    
    // Subclass 2
    class Tester extends Employee {
        String testingTools;
    
        Tester(String name, int id, String testingTools) {
            super(name, id);
            this.testingTools = testingTools;
        }
    
        void testCode() {
            System.out.println(name + " is testing using: " + testingTools);
        }
    }
    
    public class Main {
        public static void main(String[] args) {
            Developer dev = new Developer("Alice", 101, "Java, Python");

            dev.writeCode();
    
            tester.getDetails();
            tester.testCode();
        }
    }
