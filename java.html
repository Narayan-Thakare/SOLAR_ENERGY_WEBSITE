  • 13. What is the Difference Between C++ and Java?
    Answer

    1. C++ compiles directly to machine code, while Java compiles to bytecode for the JVM (Java Virtual Machine).
    2. C++ requires manual memory management, whereas Java has automatic garbage collection.
    3. Java is platform-independent due to the JVM, while C++ needs recompilation for different platforms.
    4. C++ supports pointers, while Java does not.

  • 14. Why is Java Platform Independent?
    Answer

    Java is platform-independent because its code is compiled into platform-agnostic bytecode, which can run on any system with the help of the JVM (Java Virtual Machine). The JVM acts as an interpreter that converts bytecode into machine code for different platforms, allowing the code to execute on various systems without modification.

  • 15. What is Bytecode?

  • 16. What is JVM?
    Answer

    JVM stands for Java Virtual Machine. It is an interpreter that reads bytecode line by line and converts it into machine code. This allows the code to be executed on various platforms without modification, making Java platform-independent.

  • 5. Solve the following using if-else statements: Even or Odd, Positive or Negative, Pass or Fail, Interview Eligibility, Largest of Two, Pin Verification, Rating, and Check Alphabet or Not.
    Answer

    1. Even or Odd

    
  int a = 20;
  
  if (a % 2 == 0) {
      System.out.println("This is an even number.");
  } else {
      System.out.println("This is an odd number.");
  }

    2. Positive or Negative

    
  int num = -20;
  
  if (num > 0) {
      System.out.println("This is a positive number.");
  } else {
      System.out.println("This is a negative number.");
  }

    3. Pass or Fail

    
  int percentage = 35;
  
  if (percentage >= 35) {
      System.out.println("You passed.");
  } else {
      System.out.println("You failed.");
  }

    4. Interview Eligibility

    
  int percentage = 75;
  
  if (percentage >= 65) {
      System.out.println("You can attend the interview.");
  } else {
      System.out.println("You cannot attend the interview.");
  }

    5. Largest of Two

    
  int num1 = 90;
  int num2 = 99;
  
  if (num1 > num2) {
      System.out.println("The 1st number is greater: " + num1);
  } else {
      System.out.println("The 2nd number is greater: " + num2);
  }

    6. Pin Verification

    
  int pin = 7865;
  
  if (pin == 7878) {
      System.out.println("You entered the correct pin.");
  } else {
      System.out.println("You entered the wrong pin.");
  }

    7. Rating

    
  char rating = '*';
  
  if (rating == '*') {
      System.out.println("You can give the interview.");
  } else {
      System.out.println("Please go to the mock interview.");
  }

    8. Check if Alphabet or Not

    
  char alphabet = 'g';
  
  if ((alphabet >= 'A' && alphabet <= 'Z') || (alphabet >= 'a' && alphabet <= 'z')) {

  • 13. What type of error occurs when an int data type is assigned to a byte data type using Scanner?
    Answer

    Type Mismatch Error: This error occurs because an `int` value may exceed the range of a `byte`, which can hold values from -128 to 127.

  • 13. What type of error occurs when a `short` data type is assigned to an `int` data type using Scanner?
    Answer

    Type Mismatch Error: This error occurs because a `short` value cannot be directly assigned to an `int` without explicit casting, even though `int` has a larger range than `short`.
